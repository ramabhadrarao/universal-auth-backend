Categories Module API - cURL Examples

Note: Replace YOUR_JWT_TOKEN with a valid JWT token obtained through login. Replace IDs (like CATEGORY_ID) with actual IDs from your database.

Main Category Endpoints

GET Get All Categories

Endpoint: /api/v1/categories

curl -X GET http://localhost:5000/api/v1/categories \ -H "Authorization: Bearer YOUR_JWT_TOKEN"

POST Create New Category

Endpoint: /api/v1/categories

curl -X POST http://localhost:5000/api/v1/categories \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "name": "Surgical Instruments", "description": "Tools and devices used during surgical procedures" }'

PUT Update Category

Endpoint: /api/v1/categories/:id

curl -X PUT http://localhost:5000/api/v1/categories/CATEGORY_ID \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "name": "Updated Surgical Instruments", "description": "Advanced tools and devices used during surgical procedures" }'

DELETE Delete Category

Endpoint: /api/v1/categories/:id

curl -X DELETE http://localhost:5000/api/v1/categories/CATEGORY_ID \ -H "Authorization: Bearer YOUR_JWT_TOKEN"
